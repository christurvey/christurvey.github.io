A Hitchcock-themed word guessing game. Each week there are eight unique names to solve. The names have all appeared in this week's module. Have fun guessing letter by letter!

How to play

  • Guess the letters to correctly spell a name of a person of interest to Hitchcock from this week's module
  • If you guess correctly, the letter will automatically fill in to the corresponding space(s)
  • To guess the name correctly, you must fill out all the spaces in the name
  • If you pick too many incorrect letters, the game will end up going down the drain
  • Click the "New Game" Button to start the game
